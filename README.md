# Early Warning Signals of AMOC Change in CMIP6 Models

This repository contains the code used to analyse critical slowing down (CSD) signals in historical CMIP6 simulations of the Atlantic Meridional Overturning Circulation (AMOC), as described in:

> [Full citation and DOI — to be added upon publication]

Note - some of these scripts are not operational, but rather introduce functions/ show examples of the code used. However, all the information needed to reproduce the analysis is contained in these scripts, and many of the figures can be fully reproduced using the files in data/ and the scripts in notebooks/. 

---

## Repository Contents

### Analysis scripts (scripts/)

| File | Purpose |
|------|---------|
| `EWS_functions.py` | Core CSD indicator functions: running restoring rate (λ), AR1 autocorrelation, variance, and Fourier surrogate generation |
| `EWS_wrappers.py` | High-level wrappers to combine timeseries checks and detrending with CSD indicator calculation |
| `get_ASSTI.py` | Example showing how the ASSTI fingerprint was extracted from CMIP6 SST data (final values in data/CMIP6_amoc2.nc) |
| `get_pvalues.py` | Example showing how p-values were computed by comparing observed trends to surrogate distributions |

### Notebooks (notebooks/)

| Notebook | Produces |
|----------|---------|
| `Main_Plots.ipynb` | Figures 1–6 in the main text |
| `SI_Plots.ipynb` | All supplementary figures (S3 Appendix) |
| `Table_values.ipynb` | Values for tables in S2 and S4 Appendix |

### Data files (included)

| File | Description |
|------|-------------|
| `CMIP6_amoc2.nc` | AMOC streamfunction index from Menary et al. (2020), together with the pre-computed ASSTI fingerprint for all 27 models / 133 ensemble members |
| `CMIP6_all_EWS_ppvs.nc` | Pre-computed p-values for λ, AR1, and variance (window size ws=60) |
| `CMIP6_all_EWS_ppvs_w50.nc` | As above for ws=50 |
| `CMIP6_all_EWS_ppvs_w70.nc` | As above for ws=70 |

| `data_list.txt`, `lam_list.txt`, `ar1_list.txt`, `std_list.txt` `EWS_numbers.txt`, `EWS_numbers_w50.txt`, `EWS_numbers_w70.txt` | data for tables in S4 Appendix (generated by `Table_values.ipynb`) |

## Data Availability

Large intermediate files (computed EWS arrays, surrogate distributions, and ERSSTv5 observational ensembles) are not included in this repository due to file size. These files are referenced in the notebooks via the path variables `EWS_DIR` and `OBS_DIR` defined in a configuration cell near the top of each notebook — set these to match your local paths.

The pre-computed p-value netCDF files are included for all three window sizes, so the main results can be reproduced from the notebooks without re-running the full EWS pipeline.

## Dependencies

`numpy` `scipy` `statsmodels` `xarray` `matplotlib`

Python 3.x. The notebooks also use `jupyterthemes` and `pandas` for formatting.


## Reproducing the Figures

1. Open each notebook and update the path variables in the **DATA PATHS** cell (cell 3 in each notebook):
   - `EWS_DIR`: path to the directory containing the computed EWS arrays (`CMIP6_lambdas_w60_rmean.nc`, etc.)
   - `OBS_DIR` (Main_Plots.ipynb only): path to the ERSSTv5 observational ensemble files
2. Run the notebooks.
3. Notebook order does not matter; each is self-contained.

---

## Notes on Parallel Processing

The EWS indicators were computed across 27 models × up to 10 ensemble members × 4 AMOC indices on an HPC cluster using MPI. The actual computations use only the wrapper functions in `EWS_wrappers.py`, which can be called within any parallel framework suited to your computing environment.
